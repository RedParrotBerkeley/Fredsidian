# Obsidian for assistant memory

## Question
How should Obsidian be used as part of Fred’s memory system?

## Summary
Best use is as a linked second brain, not as a day-one full replacement for assistant operational memory.

## Key findings
- markdown and links make Obsidian a strong memory substrate
- graph structure improves long-term context retrieval
- assistant memory still needs a smaller trusted operational layer

## Confidence / caveats
Concept is strong; implementation details should stay scoped and deliberate.
